# Blockchain Todo List DApp

This project is a decentralized Todo List application (DApp) built on the Ethereum blockchain. It allows users to create and manage tasks securely using smart contracts.

## Features
- Add new tasks
- Mark tasks as completed
- Data stored on blockchain
- MetaMask integration for transactions

## Tech Stack
- Solidity (Smart Contracts)
- Truffle (Development Framework)
- Ganache (Local Blockchain)
- Web3.js (Frontend Interaction)
- MetaMask (Wallet)
- HTML, CSS, JavaScript

## Modifications Made
- Updated deprecated Web3 integration
- Fixed MetaMask connection using window.ethereum
- Resolved contract loading issues
- Improved UI rendering and account handling

## How to Run
1. Start Ganache (local blockchain)
2. Deploy contracts using Truffle
3. Run frontend using `npm run dev`
4. Connect MetaMask to localhost network
